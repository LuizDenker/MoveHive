# MoveHive
Catálogo de filmes e séries, Projeto de portfólio.
Catálogo de filmes e séries que fiz para treinar HTML, CSS e JavaScript. Tem busca, cards interativos, tema escuro e conta quantas vezes a página foi aberta, Cards de filmes e séries que você pode clicar, Busca pra filtrar os títulos enquanto digita, Menu com destaque no link ativo, Botão pra voltar ao topo quando rola a página, Tema escuro e claro, Contador de visitas no rodapé e Funciona no celular também
